软件名称: Grammar Genius (文法天才)

一. 软件简介: 
使用C++语言编写的词法分析器, 能根据某一语言的正则表达式, 生成对应NFA, DFA, 最小化DFA, 词法分析源程序; 同时能选中词法分析源程序生成的可执行文件(注意: 本软件默认会调用一同打包的gcc编译器生成可执行程序， 请完成相应解压操作后再使用此功能)与待编码文件, 最后输出对应单词编码

二. 使用方法: 
***********************************************************
请完成以下操作后再运行本软件：
说明： 本操作目的为安装一个c++编译器，如果觉得操作不便可以自行使用本机编译器编译，跳过此步骤

本软件对编译生成词法分析源程序可执行程序默认采用自带的g++编译器， 但是由于其文件本身高达600+M与砺儒云单次上传限制100M， 因此将其分为mingw-w64-gcc-10.2与mingw-w64-gcc-10.2-Part02两个部分，请按照以下提示完成解压

1. 将mingw-w64-gcc-10.2-Part02.rar 压缩包解压后, 将其中的libexec文件夹与x86_64-w64-mingw32文件夹两个文件夹剪切到 Genius, 即本软件根目录下的mingw-w64-gcc-10.2文件夹中
2. 检查mingw-w64-gcc-10.2文件夹是否包含以下7个文件夹:
--bin
--i686-w64-mingw32
--include
--lib
--share
--libexec
--x86_64-w64-mingw32

3. 完成

***********************************************************
本软件主要分为以下两个子窗口
(一) 词法分析子窗口
视图说明:
视图1. [input]:   正则表达式输入视图
功能:
(1) 正则表达式输入: 支持文本框内输入与左上角菜单项"open"打开文本文件输入
(2) 词法分析源程序命名: 文本框内输入文件名即可完成命名
(3) 正则表达式保存: 支持左上角菜单项”save”保存文本框正则表达式内容
视图2. [NFA]: 正则表达式生成的NFA图
视图3. [DFA]: 正则表达式生成的DFA图
视图4. [min]: 正则表达式生成的最小化DFA图

视图5. [run]: 单词编码视图
功能: 
(1) 浏览文件夹选中词法分析源程序生成的可执行程序: 点击"open"
(2) 输入待编码源程序:支持文本框输入与浏览文件打开


(二) 文法分析子窗口
视图说明:
视图1. [Input] : 文法规则输入视图
功能:
(1) 文法规则输入: 支持文本框内输入与左上角菜单项"open"打开文本文件输入
(2) 文法规则保存: 支持左上角菜单项”save”保存文本框正则表达式内容
(3) 展示first集合与follow集合, 是否为文法冲突情况等信息

视图2. [Map]: 映射表展示视图, 展示文法规则单词-常用ascii字符 1对1 映射表与归约式简写前后的映射
视图3. [LR(0)DFA]: LR(0)DFA展示视图, 展示邻接矩阵形式的LR(0)DFA图
视图4. [SLR(1)]: SLR1分析表展示视图, 展示SLR1分析表

视图5. [Run]: 文法分析视图
功能:
(1) 支持文本框输入或者文件浏览输入内容. 若内容为源代码, 则需要另外打开一个对应词法分析可执行程序转成单词编码输入, 如果为单词编码, 直接运行即可进行分析
(2) 若语句最终被接收, 则输出文法分析过程与语法树; 若不被接收, 则弹出警告信息

三. 警告信息说明
以下为常见报错符号及其信息:
E1: Null string received!
读取转化后, 用来处理的单词编码输入为空串
E2: Please browse to open a file or directly enter text in the text box
[Run]视图文本输入框为空
E3: Please select the input mode (lex file/source file) 
[Run]视图没有选择输入模式
E4: Please switch to SOURCE mode! 
[Run]视图选择可执行程序时没有切换到SOURCE模式
E5: Please select the path for the EXE 
[Run]视图文本框输入的为源代码时, 没有请选择可执行程序的路径
E6: Please generate the SLR1 parsing table before performing syntax analysis. 
[Run]视图, 文法分析前没有生成SLR1就来进行语法分析
E7: The statement you entered is not grammatically correct and therefore cannot be accepted.
run
[Run]视图, 输入不符合语法, 即不接受
E8: Please select the PROPER input mode (lex file/source file)
[Run]视图选择的输入模式与文本框文本不匹配
