一.编译环境:
本项目在visual studio 2022集成开发环境与qt 5.14下完成,编译运行前请配置好相应环境
MSVC版本：1936 

二.编译步骤:
***********************************************************
请完成以下操作后再运行源程序：
说明： 
本软件对编译生成词法分析源程序可执行程序默认采用自带的g++编译器， 但是由于其文件本身高达600+M与砺儒云单次上传限制100M， 因此将其分为mingw-w64-gcc-10.2与mingw-w64-gcc-10.2-Part02两个部分，请按照以下提示完成解压

1. 将mingw-w64-gcc-10.2-Part02.rar 压缩包解压后, 将其中的libexec文件夹与x86_64-w64-mingw32文件夹两个文件夹剪切到 Genius, 即本软件根目录下的mingw-w64-gcc-10.2文件夹中
2. 检查mingw-w64-gcc-10.2文件夹是否包含以下文件夹:
--bin
--i686-w64-mingw32
--include
--lib
--share
--libexec
--x86_64-w64-mingw32

3. 完成

***********************************************************
1. 准备好visual studio 2010版本以上编译器，找到“项目源程序\Genius” 下的Genius.sln文件并点击打开

2. 使用visual studio 2010版本以上编译器点击 Genius.sln 运行

3. 在visual studio 2010版本以上编译器界面上方点击调试
